Vérifiez que l'adresse email est correcte. C'est sur cette adresse email que vous sera envoyée l'autorisation ESTA.

En cas de problème, merci d'envoyer un email à l'adresse suivante: contact@france-esta.fr

Passeport Ouvert:

Renseignez votre nom de famille tel qu'il figure dans votre passeport dans le champ Nom de famille.

Dans le cas où votre nom de famille contient des caractères non anglais (accent par exemple), merci de consulter le bas de la page contenant la photo et inscrivez le nom de famille tel qu’il figure entre les chevrons
(<(NAT)NOM DE FAMILLE<<).

Vous pouvez également nous envoyer une copie scannée de votre passeport si vous souhaitez obtenir de l'aide. Pour nous contacter, merci d'envoyer un email à l'adresse suivante: contact@france-esta.fr

Passeport Ouvert:

Renseignez TOUS les prénoms exactement tels qu'ils apparaissent sur votre passeport dans le champ Prénoms.

Si un deuxième prénom est inclus dans le champ Prénom de votre passeport, vous devez également l'inclure ici. Si vous n'avez pas de prénom, merci de taper les lettres FNU qui signifient Prénom inconnu.

Dans le cas où votre prénom contient des caractères non anglais (accent par exemple), merci de consulter le bas de la page contenant la photo et inscrivez le prénom tel qu’il figure entre les chevrons
(<< PRENOM <<).

Vous pouvez également nous envoyer une copie scannée de votre passeport si vous souhaitez obtenir de l'aide. Pour nous contacter, merci d'envoyer un email à l'adresse suivante: contact@france-esta.fr

Ville :

Renseignez la ville, le chef-lieu, le village ou le hameau où vous êtes né.

Si vous ne connaissez pas la ville où vous êtes né, entrez UNKNOWN.

Informations - Site non officiel

Le site internet france-esta.fr n'est pas affilié au site officiel du gouvernement Américain, de quelques manières que ce soit. Il est géré par une équipe de professionnels, avec une longue expérience en matière de formalité pour voyager aux USA. Notre tarif pour demander un ESTA est de 89,90 €, incluant les 14 $ demandés par le gouvernement Américain.
